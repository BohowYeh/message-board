# 練習
建造一個網路留言板，後端使用python，將留言者的各項資料從db資料庫存入或取出顯示在前端。

# 說明
一般使用者可以在留言板留言，admin做為管理者可以登入後台刪除或更改留言。各個使用者的留言資料都會存進資料庫db
